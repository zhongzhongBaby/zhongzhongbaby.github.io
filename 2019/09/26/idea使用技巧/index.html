idea使用技巧

Sep 26, 2019工具Hits

以下是我在idea使用过程中总结的一些小技巧，可以帮助提高开发效率，对idea不熟悉的同学，可以作为参考。

idea使用技巧

一.快捷键

idea的快捷键网上一搜一大堆 ，这是我自己用得很顺手的快捷键。

  • Ctrl+E，最近的文件
  • Alt+Insert，可以生成构造器/Getter/Setter等
  • Ctrl+Alt+T，可以把代码包在一个块内，例如：try/catch
  • Ctrl+Alt+L，格式化代码
  • Ctrl+D，复制行
  • Ctrl+H，显示类结构图（类的继承层次）
  • Ctrl+ALT+H，显示方法被谁调用
  • Ctrl+Alt+left/right，返回至上次浏览的位置 (很顺手)
  • alt+7 类的所有方法
  • ctrl+shift+z 恢复撤销
  • ctrl+shift+k git push
  • ctrl + / 单行注释
  • Alt+Up/Down，在方法间快速移动定位
  • Ctrl+Shift+U，大小写转化
  • ctrl+shift+上下键 行上下移动
  • psvm + tab键 生成main方法
  • sout System.out.print()

二.几个好用的功能

1.特定文件版本历史

2.todo注释，快速定位待完善功能，在开发初期可以快速构建伪代码

3.编译单个类，无需重启服务

4.粘贴板历史 不需要来回粘贴复制

5.方法名字批量更换

6.代码辅助校验

7.对比行数

三.常用设置

1.滚动改变字体大小

2.自动导包（unambiguous）和 去除没有用到的包

3.方法被调用之后 仍然提示Method *** is never used

File -> Invalidate Caches

4.设置控制台可以输入

打开idea配置文件（…….bin/idea64.exe.vmoptions）添加
-Deditable.java.test.console=true
重启
